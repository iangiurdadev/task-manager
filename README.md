# Task Manager Java

A simple command-line task manager built in Java.

## Features

- Create tasks
- List all tasks
- Mark tasks as completed
- Delete tasks
- Automatic task ID generation

## Technologies

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collections Framework

## Project Structure

```text
Main.java
Task.java
TaskService.java
```

## How to Run

Compile the project:

```bash
javac *.java
```

Run the application:

```bash
java Main
```
